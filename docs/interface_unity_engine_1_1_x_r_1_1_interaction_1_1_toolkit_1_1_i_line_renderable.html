bool UnityEngine.XR.Interaction.Toolkit.ILineRenderable.GetLinePoints ( ref Vector3[] linePoints,
out int numPoints )

Gets the polygonal chain represented by a list of endpoints which form line segments to approximate the curve. Positions are in world space coordinates.

Parameters
linePointsWhen this method returns, contains the sample points if successful.
numPointsWhen this method returns, contains the number of sample points if successful.
Returns
Returns true if the sample points form a valid line, such as by having at least two points. Otherwise, returns false.

Gets the current ray cast hit information, if a hit occurs. It returns the world position and the normal vector of the hit point, and its position in linePoints.

Parameters
positionWhen this method returns, contains the world position of the ray impact point if a hit occurred.
normalWhen this method returns, contains the world normal of the surface the ray hit if a hit occurred.
positionInLineWhen this method returns, contains the index of the sample endpoint within the list of points returned by GetLinePoints where a hit occurred. Otherwise, a value of 0 if no hit occurred.
isValidTargetWhen this method returns, contains whether both a hit occurred and it is a valid target for interaction.
Returns
Returns true if a hit occurs, implying the ray cast hit information is valid. Otherwise, returns false.
