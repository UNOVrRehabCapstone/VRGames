◆ Initialize()

override bool Unity.XR.Oculus.OculusLoader.Initialize ( )
virtual

Initialize the loader. This should initialize all subsystems to support the desired runtime setup this loader represents.

This is the only method on XRLoader that Management uses to determine the active loader to use. If this method returns true, Management locks this loader as the XRManagerSettings.activeLoader and and stops fall through processing on the XRManagerSettings.loaders list of current loaders.

If this method returns false, XRManagerSettings continues to process the next loader in the XRManagerSettings.loaders list, or fails completely when the list is exhausted.

Returns
Whether or not initialization succeeded.
