Detailed Description

Reads hand & HMD pose data from the DataSources, and copies them onto the given OVRSkeleton. Can also provide more sophisticated control of the mesh rendering, to keep hand meshes visible when tracking is lost.

This class is not required if you are using the HandSkeletonVisual to render the hand.
