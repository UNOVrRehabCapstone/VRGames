◆ MenuActionState

enum UnityEditor.ProBuilder.MenuAction.MenuActionState

Determines whether the menu item is visible, and if visible, whether it is enabled.

Enumerator
Hidden 

The button is not visible in the toolbar.

Visible 

The button is visible in the toolbar.

Enabled 

The button (and by proxy, the action it performs) are valid given the current selection.

VisibleAndEnabled 

Button and action are both visible in the toolbar and valid given the current selection.

◆ PerformAction()

ActionResult UnityEditor.ProBuilder.MenuAction.PerformAction ( )

Performs the action for this menu item. Use PerformActionImplementation to implement the action. Calling this method triggers the onPerformAction event.

Any new action classes that derive from this base class must also use the PerformAction method to register the new action in the Undo call stack.

Returns
A new ActionResult with a summary of the state of the action's success.
