Detailed Description

Used in conjunction with HandGrabInteractable or DistanceHandGrabInteractable, the HandGrabPoint defines the local point in an object to which the grip point of the hand should align. It can also contain information about the final pose of the hand for perfect alignment as well as a surface that indicates the valid positions for the point.
