<img src="images/icons/Selection_Ring.png" alt="Select Edge Ring icon"/> Select Edge Ring

The Select Edge Ring action selects a ring from each selected edge. An edge ring is a series of edges which share faces but are not directly connected.

‍Note: This action only works on quads (four-sided polygons), and is only available in edge mode.

‍Tip: You can also use this action with the Alt/Opt+R shortcut, or from the ProBuilder menu (Tools > ProBuilder > Selection > Select Ring).
