Detailed Description

Raises events when an object is rotated relative to a provided transform. Rotated events will be raised when the rotation exceeds a provided angle threshold, in degrees. Events are raised only once per directional sweep, so if an event was fired while angle was increasing, the next must fire while angle decreases.
