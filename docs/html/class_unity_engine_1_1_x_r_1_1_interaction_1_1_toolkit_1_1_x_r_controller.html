Detailed Description

Interprets feature values on a tracked input controller device from the XR input subsystem into XR Interaction states, such as Select. Additionally, it applies the current Pose value of a tracked device to the transform of the GameObject.

It is recommended to use ActionBasedController instead of this behavior. This behavior does not need as much initial setup as compared to ActionBasedController, however input processing is less customizable and the Inputs.Simulation.XRDeviceSimulator cannot be used to drive this behavior.

See also
XRBaseController, ActionBasedController
