Detailed Description

Abstract base class from which all interactor behaviours derive. This class hooks into the interaction system (via XRInteractionManager) and provides base virtual methods for handling hover and selection.

You can use this to change the effective movement type of an Interactable for different Interactors. An example would be having an Interactable use XRBaseInteractable.MovementType.VelocityTracking so it does not move through geometry with a Collider when interacting with it using a Ray or Direct Interactor, but have a Socket Interactor override the movement type to be XRBaseInteractable.MovementType.Instantaneous for reduced movement latency.
