Detailed Description

InteractorDriverGroup coordinates between a set of InteractorDrivers to determine which InteractorDriver(s) should be enabled at a time.

By default, InteractorDrivers are prioritized in list order (first = highest priority). InteractorDrivers can also be prioritized with an optional IInteractorComparer

◆ InteractorDriverGroupStrategy

enum Oculus.Interaction.InteractorDriverGroup.InteractorDriverGroupStrategy

InteractorGroupStrategy defines how an InteractorDriverGroup coordinates hovering between InteractorDrivers when no InteractorDriver is yet selected. Upon selection, all other drivers are disabled.

Enumerator
PRIORITY 

Until selecting, reevaluate drivers in priority order.

FIRST 

Until hovering, reevaluate drivers in priority order.

MULTIPLE 

Until selecting, allow multiple hovered drivers.
