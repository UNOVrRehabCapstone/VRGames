Detailed Description

Represents an edge connecting two vertices.

This might point to an index in the vertices (local) or the ProBuilderMesh.sharedVertices (common) array. The ProBuilder terminology "local" and "common" refers to whether this is an index from the list of all vertices in the ProBuilderMesh or an index from the list of only shared vertices.
