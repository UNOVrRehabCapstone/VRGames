Detailed Description

A single matched DeviceRequirement.

Links the control that was matched with the respective device requirement.

Property Documentation

◆ control

InputControl UnityEngine.InputSystem.InputControlScheme.MatchResult.Match.control
get

The control that was match from the requirement's DeviceRequirement.controlPath

This is the same as device if the control path matches the device directly rather than matching a control on the device.

Note that while a control path can match arbitrary many controls, only the first matched control will be returned here. To get all controls that were matched by a specific requirement, a manual query must be performed using InputControlPath.

If the match failed, this will be null.

◆ device

InputDevice UnityEngine.InputSystem.InputControlScheme.MatchResult.Match.device

The device that got matched.

If a specific control on the device was matched, this will be InputControl.device or control. If a device was matched directly, this will be the same as control.
