Detailed Description

Modifies the active state of a list of GameObjects, as well as the enabled state of a list of components, from the Active field of the given IActiveState. The component will only activate/enable dependants that were active/enabled during Start() lifecycle phase.

These need to be updated in batch or else we could get inconsistent behaviour when multiple of these are in a scene.
