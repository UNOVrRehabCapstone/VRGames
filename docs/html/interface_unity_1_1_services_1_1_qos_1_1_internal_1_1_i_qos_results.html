Task< IList< QosResult > > Unity.Services.Qos.Internal.IQosResults.GetSortedQosResultsAsync ( string service,
IList< string > regions )

Gets sorted QoS measurements the specified service and regions.

GetSortedQosResultsAsync doesn't consider the returned regions until applying the services and regions filters.

If you specify regions, it only includes those regions.

Parameters
serviceThe service to query regions for QoS. GetSortedQosResultsAsync only uses measures regions configured for the specified service.
regionsThe regions to query for QoS. If not null or empty, GetSortedQosResultsAsync only uses regions in the intersection of the specified service and the specified regions for measurements.
Returns
Returns the sorted list of QoS results, ordered from best to worst.
