UnityEditor.ProBuilder.TooltipContent.TooltipContent ( string title,
string summary,
params char[] shortcut )

Creates a new tooltip with a title, a summary, and an optional array of characters for the shortcut.

To specify modifier keys, use the Windows control keys. ProBuilder manages switching to Linux control keys for the macOS and Linux versions of the Unity Editor.

Parameters
titleThe header text for this tooltip.
summaryThe body of the tooltip text. This should be kept brief.
shortcutA set of keys to be displayed as the shortcut for this action.
