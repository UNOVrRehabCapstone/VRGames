Detailed Description

Represents an edge composed of both the local index and the common index.

Use this to compare vertex indices that are coincident. Coincident vertices share the same coordinate space, but are separate values in the vertex array. ProBuilder tracks these coincident values in the ProBuilderMesh.sharedVertices array. A "common" (also called "shared") index is the index of a vertex in the sharedVertices array.
