Detailed Description

Wraps around values provided by input actions.

This is a wrapper around InputAction.CallbackContext chiefly for use with GameObject messages (i.e. GameObject.SendMessage(string,object)). It exists so that action callback data can be represented as an object, can be reused, and shields the receiver from having to know about action callback specifics.
